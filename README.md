# inv_bubble_opt

Tools for identifying material parameters from bulge inflation test data.

# Getting started

Clone this repo

```
git clone https://github.com/cjekel/inv_bubble_opt.git
```

and install the invbubble library

```
cd inv_bubble_opt
pip install invbubble_py_source/
```
